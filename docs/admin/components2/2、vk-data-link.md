# 2、link 外部超链接

### template 使用方式

```html
<vk-data-link href="https://ext.dcloud.net.cn" text="https://ext.dcloud.net.cn"></vk-data-link>
```
### API

### 属性

| 参数             | 说明                           | 类型    | 默认值  | 可选值 |
|------------------|-------------------------------|---------|--------|-------|
| href            | 目标url地址 | String  | - | -  |
| text          | 显示的文本| String  | - | - |
| show-under-line          | 是否显示下划线  | Boolean  | true | false  |
| color      | 文本颜色 | String  | #999999 | -  |
| font-size         | 字体大小 | Number| String  | 13 | -  |
