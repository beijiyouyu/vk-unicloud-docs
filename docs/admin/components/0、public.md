# 0、公共属性

| 参数           | 说明             | 类型   | 默认值 | 可选值 |
|---------------|------------------|--------|-------|-------|
| key           | 字段名            | String | -    | -      |
| title         | 字段显示的名称     | String | -    | -      |
| type          | 组件类型          | String | -    | -      |
| placeholder   | 占位符            | String | -    | -      |
| width         | 宽度              | Number | -    | -      |
| disabled      | 是否禁用          | Boolean | false | true  |
| clearable     | 是否可以清空选项   | Boolean | true | false  |
| tips          | 下方的提示         | String | - | -  |
