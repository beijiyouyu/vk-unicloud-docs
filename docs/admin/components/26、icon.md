# 26、icon 图标选择

### 万能表单使用方式

```js
{ key:"icon", title:"图标", type:"icon" },
```

### 属性

[点击查看『公共属性』](https://vkdoc.fsq.pub/admin/components/0%E3%80%81public.html)

### 组件属性

| 参数             | 说明                           | 类型    | 默认值  | 可选值 |
|------------------|-------------------------------|---------|--------|-------|
| multiple            | 是否可多选 | Boolean  | false | true  |
| multipleLimit          | 最大多选数量 | Number  | - | -  |
