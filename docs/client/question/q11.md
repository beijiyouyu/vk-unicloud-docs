# 11、云函数请求报跨域错误

### 解决办法：

将你用到的域名全部加入到unicloud web控制台的跨域白名单

如:
- 自己的域名1
- 自己的域名2

同时顺便把localhost也加入了

- localhost:8080
- localhost:8081
- localhost:8082
- localhost:8083
- localhost:3000
- localhost:3001
- localhost:3002
- localhost:3003

### 步骤：

- 1、 进入unicloud web控制台 https://unicloud.dcloud.net.cn/domain

- 2、 点击`跨域配置` - `新增域名`

- 3、 输入域名
